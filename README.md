It is a weather fetching agent , if user  type a city then first it will fetch the coordinates of the city using a geocoding api called Nominatim(openstreetmap) then
using those coordinates it will fetch the weather details like temp,windspeed etc .It can store all the searches the user has ever searched in a json file , user can 
view the history anytime using simple commands and can also delete all the recent searches.The core idea of this repo is API integration and handling error handling, json handling,multi file project structure and modular functions.
